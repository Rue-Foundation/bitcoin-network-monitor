What's going on here?
This page is a client-side tool you can run by yourself to verify we are behaving honestly.
Keep this monitoring page open if you like, we won't disappoint you!

Loading the page could take a while since lot of things are going on here on your client side.
This is what it is doing to verify our honesty:
  • downloading from a public Bitcoin node, via insight API, the last blocks along with the confirmed transactions
  • checking if any Oraclize proof reference is there, in which case the full proof is downloaded from a public IPFS gateway
  • verifying each proof is actually valid
